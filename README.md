\## Project Overview

This project is an Android application developed using Android Studio for SIT305 Task 2.1P.  

The application functions as a travel companion converter, allowing users to convert values across different categories such as temperature, currency, and distance.



The aim of the app is to provide a simple, efficient, and user-friendly tool that can assist travellers in performing quick conversions without relying on external tools.



\---



\## Features

\- Multiple conversion categories (e.g., Temperature, Currency, Distance)

\- Dropdown selection for source and destination units

\- User input field for entering values

\- Convert button to process calculations

\- Display of converted results

\- Simple and clean user interface



\---



\## How the Application Works

The user begins by selecting a conversion category from the dropdown menu.  

They then choose the source unit and destination unit. After entering a numerical value, the user presses the \*\*Convert\*\* button.  



The application processes the input and displays the converted result instantly on the screen.
